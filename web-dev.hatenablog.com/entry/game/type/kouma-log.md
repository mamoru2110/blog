タイピング紅魔郷の変更履歴です。


## お願い
ＰＣに古いバージョンがインストールされている場合、最新版のダウンロードとインストールをして頂けると嬉しいです。ゲームのデータは保持されます。


## 変更履歴
### v1.0.1（2019.09.18）
- **更新**：環境を Windows10 64bit に限定（Win7サポート切れ）
- **更新**：インストーラーを変更（Win10のみ動作）
- **更新**：ローマ字の大文字入力をサポート（Caps Lock 時の入力）
- **修正**：音量調整の内部処理（Unity AudioMixer を使用）
