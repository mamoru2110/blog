## 歌詞
```
A
闇が満ちるとき、私は踊るの
変わらない、時の流れ、確かめるように

人が眠るとき、運命は動くの
終わらない、時の中で、別れがはじまる


B
そう、この笑顔もいつか
涙に変わる、分かってたけど

あの笑顔が最後、戻れないこと
分かっているけど、、まだ

　
C
祈りはしない、届かないから
忘れはしない、あなたのことを

願いもしない、叶わないから
忘れはしない、それだけでいい
```
